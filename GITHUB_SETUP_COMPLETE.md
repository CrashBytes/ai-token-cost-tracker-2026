## 🛠️ **How to Use**

### For Maintainers

**Create New Issue with Labels:**
```bash
gh issue create --title "Add Gemini Flash support" \
  --body "Support for Google's Gemini Flash model" \
  --label "feature,google-ai,pricing"
```

**Label Existing Issue:**
```bash
gh issue edit 123 --add-label "good-first-issue,help-wanted"
```

**View All Labels:**
```bash
gh label list
```

### For Contributors

**Find Good First Issues:**
- Visit: https://github.com/CrashBytes/ai-token-cost-tracker-2026/labels/good-first-issue
- Or: `gh issue list --label "good-first-issue"`

**Read Contribution Guide:**
- See: [CONTRIBUTING.md](CONTRIBUTING.md)
- Includes setup, testing, and PR guidelines
